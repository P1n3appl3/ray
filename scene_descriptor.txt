data types are:

f means float
v(f, f, f) is vector
u is unsigned
b is bool
s is string
bg is a texture
Texture can be:
    image(s)
    checkered[3d](Texture, Texture)
    perlin(Kind(simple enum))
    solid(v)
Material can be:
    diffuse(Texture)
    light(Texture)
    specular(albedo: v, fuzz: f)
    dielectric(tint: v, refractive_index: f)
Object can be:
    prism(size: v, mat: s)
    sphere(pos: v, radius: f, mat: s)
    rect(axis_a: a, axis_b: a, a0: f, b0: f, a1: f, b1: f, mat: s)
    mesh(obj: s, scale: f, mat: s)
    flip(Object)
    rotate(obj: Object, axis: a, degrees: f)
    translage(obj: Object, pos: v)

and the file looks like this:

scene: ( // implicit
width: u
height: u
samples: u
bounces: u
showbg: b //default true
bg: Texture //default white/blue gradient
camera: (
    pos: v
    look_at: v //default (0, 0, 0)
    up: v //default (0, 1, 0)
    fov: f //default 90
    aperture: f //default 0
)
materials: {
    "name" Material()
    "other_name" Material()
    ...
}
entities: [
    Object(),
    ...
]
